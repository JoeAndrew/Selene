# Séléné

**Séléné** is a lightweight and versatile automation tool, driven by end user customized **Lua scripts** and acting on **MQTT data**.

Dependencies
------------

  -	**PAHOc** can be found on [its website](https://eclipse.org/paho/clients/c/).
